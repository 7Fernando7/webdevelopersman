HTML a Secas

En todo documento HTML se distinguen en dos tipos de elementos

Etiquetas: Indican como se ha de organizar el contenido del documento

  • Están delimitadas por los caracteres "(< y >)" y pueden tener una gemela de cierre delimitando as su area de influencia.
  • Estas etiquetas pueden tener atributos, aunque la tendencia en HTML5 es eliminar el uso de estos atributos utlizando las hojas de estilo CSS.

Contenido:Es información contenida en el documento

Ejemplo:

<p class="normal">Esto es un párrafo</p>

Ejemplo de una página web:

Estructura de un documento HTML

<HTML>

<HEAD>

<TITLE> Título de la página </TITLE>

</HEAD>

<BODY>

Contenido de la página

</BODY>

</HTML>

La etiqueta <HTML> ... </HTML> indica el comienzo y el final del documento HTML

Un documento HTML básico tiene una cabecera y un cuerpo.

Cabecera: <HEAD> ... </HEAD>

Cuerpo: <BODY> ... </BODY>

<HTML>: indica el comienzo y el final de un docum/ento HTML. Todo lo que se coloque fuera de la etiqueta <html> se ignora, salvo alguna excepción.

<HEAD>: es la cabecera del documento. La cabecera contiene información sobre el propio documento HTML, el título y el idioma de la página. Los contenido indicados en la cabecera no son visibles para el usuario, con la excepción de la etiqueta <title>.

</BODY>: es el cuerpo del documento HTML. El cuerpo encierra todos los contenidos que muestran al usuario(párrafos de texto, imágenes, tablas).

EL TEXTO

La etiqueta <p> se utiliza para representar un párrafo.

<p>

Esto es un texto de párrafo.

</p>

Etiquetas de linea para el texto:

ETIQUETAS DE ESTILO FÍSICO

Especifican un cambio concreto en el tipo de letra

  • Texto en negrita: <B>...</B>
  • Texto en cursiva: <I>... </I>
  • Texto subrayado: <U>... </U>
  • Texto monoespaciado: <TT>... </TT>
  • Texto tachado: <STRIKE>... </STRIKE>
  • Aumentar el tamaño del texto: <BIG>... </BIG>
  • Disminuir el tamaño del texto: <SMALL>... </SMALL>
  • Texto en sub índice: <SUB>... </SUB>
  • Texto en super índice: <SUP>... </SUP>

Desaprobadas

ETIQUETAS DE ESTILO LOGICO

Caracterizan un trozo de texto destacándolo en el contexto
  • Texto destacado: <EM>...</EM>
  • Texto importante: <STRONG>... </STRONG>
  • Citas y referencias: <CITE>... </CITE>
  • información sobre el autor: <ADDRESS>... </ADDRESS>
  • Definición de un término: <DFN>... </DFN>
  • Fragmento de código de programa: <CODE>... </CODE>
  • Ejemplo de programa: <SAMP>... </SAMP>
  • Variables, argumentos de un programa: <VAR>... </VAR>

LISTAS

La etiqueta <ul> se utiliza para crear una lista desordenada.

<ul>

<li>HTML</li>

<li>CSS</li>

<li>JavaScript</li>

</ul>

  • HTML
  • CSS
  • JavaScript

Listas no enumeradas: <UL> ... </UL>

Atributos de esta etiqueta

El elemento termina

al encontrar la etiqueta final</LI> (no es obligatoria)

al encontrar otra etiqueta inicial <LI>

al encontrar la etiqueta final de lista

La etiqueta <ol> se utiliza para crear una lista desordenada.

<ol>

<li>HTML</li>

<li>CSS</li>

<li>JavaScript</li>

</ol>
  1. HTML
  2. CSS
  3. JavaScript

Listas enumeradas: <OL> ... </OL>

Atributos de esta etiqueta

Tipo de enumeración: type="1"|"A"|"a"|"I"|"i"

Valor inicial: start="numero"

La etiqueta <dd> se utiliza para crear una lista desordenada.

<dl>

<dt>Chocolate</dt>

<dd>Elaborado a base de cacao</dd>

<dt>Caramelo</dt>

<dd>Elaborado a base de azucar</dd>

</dl>

Chocolate
Elaborado a base de cacao
Caramelo
Elaborado a base de azucar

Lista de descripciones: <DL> ... <DL>

Cada elemento de la lista está formado por dos partes

  • El término que queremos describir: <DT>
  • La descripción de dicho término: <DD>

Ambos elementos terminan

  • al encontrar la etiqueta final: (<DT> o <DD>)
  • al encontrar otra etiqueta inicial: (<DT> o <DD>)
  • al encontrar la etiqueta final de lista

Enlaces

<a href="URI">Texto</a>

  • Enlaces a otras páginas <a href="URI">Texto</a>
  • La dirección de la página enlazada se resulve teniendo en cuenta el valor de la dirección URL del atributo href.
  • El Texto es el elemento activo a través del cual el navegador puede acceder a la página enlazada
  • El color del elmento activo Texto varía en función de si la página ya ha sido visitada por el navegador o no
  • Si se accede a la página enlazada, el navegador se situá al comienzo de la misma

Anclas

<a name="nombre">Texto</a>

Las "anclas" so marcas específicas incluidas en el interior de un documento para indicar el comienzo de una sección o bloque de contenido:

<a name="nombre">Texto</a>

  • El "nombre" tiene que ser unico para cada marca
  • La etiqueta no tiene ningún efecto visible sobre el texto

<a href="#nombreAncla">Texto</a> → <a name="nombreAncla">Texto</a>

Enlaces a puntos internos de otro documento

  • <a href="URI">Texto</a>
  • Donde nombre es un "ancla" en el documento de dirección URI
  • Si se accede a la página enlazada, el navegador se situán en el punto indicado por la marca "nombre"

<a href="#nombrePagina#nombreAncla">Texto</a> → <a name="nombreAncla">Texto</a>

Imagenes

<IMG src="URL"/>

  • Inclusión de una imagen:<IMG src="URL"/> (no tiene etiqueta de cierre)
  • La dirección de la imagen enlazada se resulve teniendo en cuenta el valor de la dirección del atributo "src". Los formatos de imagen usuales son GIF, JPG, o PNG.

    • Atributos de esta etiqueta

      • Alternativa para navegadores no gráficos o error en la localización de la imagen: alt="texto".
      • Muy buenos resultados en el posicionamiento junto al atributo tittle="titulo imagen".
      • Borde: border="pixels"
      • Anchura: width="longitud"
      • Altura: height="longitud"

      Es muy frecuente utilizar imágens activas que enlazan a otros documentos

      Esto se consigue combinando las etiquetas <A...> e <IMG...> de la seguiente forma:

      <a href="URI"><IMG src="URI-imagen"/><A...>
      • En este caso se incluye un borde azul de 1 pixel de ancho, para quitar este borde hay que incluir el atributo border="0" en la etiqueta IMG.
